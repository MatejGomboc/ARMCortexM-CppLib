# ARMCortexM-CppLib

CMSIS-like C++ library for ARM Cortex-M microcontrollers

## Supported Architectures

- ARM Cortex-M0
- ARM Cortex-M0+
- ARM Cortex-M1 (WIP)
- ARM Cortex-M3 (WIP)
